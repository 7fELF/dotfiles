# dotfiles

## Requirements
  - wget or curl
  - sudo

## Quick install command
```
curl -sSL https://raw.githubusercontent.com/antoinegergy/dotfiles/master/install.sh | sh
```
or:
```
wget -qO- https://raw.githubusercontent.com/antoinegergy/dotfiles/master/install.sh | sh
```
